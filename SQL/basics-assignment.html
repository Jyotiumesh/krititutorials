Create one database with the name of aptech
use and create this below student table insert 1 to 3 records from By specifying column names method 4 and 5 from Without specifying column names method and 6 to 8 From Multiple values method-


Solution:-

create database aptech;

use aptech;

create table student(No int,Name varchar(10),Age int,Department varchar(10),DOA date,Fee int,Gender varchar(10));
